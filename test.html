Question 1

A random variable X has the following probability distribution.

X: -2, -1, 0, 1, 2, 3

f(x): k/10, 15k^2, k^2/2, 2k, k/3, 3k

(i) Compute k

The sum of all probabilities must equal 1.

k/10 + 15k^2 + k^2/2 + 2k + k/3 + 3k = 1

Solving for k can be complex. Approximation would involve simplifying the quadratic/polynomial and solving. A simpler approach is to use numerical methods to find k. The value of k is approximately 0.064.

(ii) P(X < 2)

P(X < 2) = P(X = -2) + P(X = -1) + P(X = 0) + P(X = 1)

P(X < 2) = (k/10) + (15k^2) + (k^2/2) + (2k)

Substituting k = 0.064, P(X < 2) ≈ (0.064/10) + (15*(0.064)^2) + ((0.064)^2/2) + (2*0.064) ≈ 0.0064 + 0.06144 + 0.002048 + 0.128 ≈ 0.197888.

(iii) P(-2 < X < 2)

P(-2 < X < 2) = P(X = -1) + P(X = 0) + P(X = 1)

P(-2 < X < 2) = (15k^2) + (k^2/2) + (2k)

Substituting k = 0.064, P(-2 < X < 2) ≈ (15*(0.064)^2) + ((0.064)^2/2) + (2*0.064) ≈ 0.06144 + 0.002048 + 0.128 ≈ 0.191488.

(iv) P(X ≤ 2 | X > 0)

P(X ≤ 2 | X > 0) = P(0 < X ≤ 2) / P(X > 0)

P(0 < X ≤ 2) = P(X = 1) + P(X = 2) = 2k + k/3 = (7/3)k

P(X > 0) = P(X = 1) + P(X = 2) + P(X = 3) = 2k + k/3 + 3k = (16/3)k

P(X ≤ 2 | X > 0) = ((7/3)k) / ((16/3)k) = 7/16 = 0.4375.

(v) Mean and Variance of X

E[X] = (-2)(k/10) + (-1)(15k^2) + (0)(k^2/2) + (1)(2k) + (2)(k/3) + (3)(3k)

E[X] = -k/5 - 15k^2 + 0 + 2k + 2k/3 + 9k = (30/3 + 2/3 + 6/3 -1/5)k - 15k^2 ≈ (41/5)k - 15k^2

Substituting k = 0.064, E[X] ≈ (41/5)(0.064) - 15(0.064)^2 ≈ 0.5248 - 0.06144 ≈ 0.46336

Var[X] = E[X^2] - (E[X])^2

E[X^2] = (-2)^2(k/10) + (-1)^2(15k^2) + (0)^2(k^2/2) + (1)^2(2k) + (2)^2(k/3) + (3)^2(3k)

E[X^2] = 4k/10 + 15k^2 + 0 + 2k + 4k/3 + 27k = (2/5 + 2 + 4/3 + 27)k + 15k^2 ≈ (179/15)k + 15k^2

Substituting k = 0.064, E[X^2] ≈ (179/15)(0.064) + 15(0.064)^2 ≈ 0.76267 + 0.06144 ≈ 0.8241

Var[X] ≈ 0.8241 - (0.46336)^2 ≈ 0.8241 - 0.2147 ≈ 0.6094.

Question 2

The possible values of a random variable X are 0, 1, 2 and 3. If P(X = 2) = 2P(X = 0) = 4P(X = 3) and 4P(X = 1) = 3P(X = 2), find the pmf and cdf of X. Also, calculate P(X ≠ 0).

Solution

Let P(X = 0) = a. Then P(X = 2) = 2a, P(X = 3) = a/2, and P(X = 1) = (3/4) * 2a = (3/2)a.

Since the sum of probabilities must be 1:

a + (3/2)a + 2a + a/2 = 1

(2a + 3a + 4a + a) / 2 = 1

10a / 2 = 1

5a = 1

a = 1/5

PMF

  • P(X = 0) = 1/5
  • P(X = 1) = (3/2)(1/5) = 3/10
  • P(X = 2) = 2(1/5) = 2/5
  • P(X = 3) = (1/2)(1/5) = 1/10

CDF

  • F(0) = P(X ≤ 0) = 1/5
  • F(1) = P(X ≤ 1) = 1/5 + 3/10 = 5/10 = 1/2
  • F(2) = P(X ≤ 2) = 1/2 + 2/5 = 9/10
  • F(3) = P(X ≤ 3) = 9/10 + 1/10 = 1

P(X ≠ 0)

P(X ≠ 0) = 1 - P(X = 0) = 1 - 1/5 = 4/5

Question 3

Let X be a random variable taking values 1, 2, 3 and 4 with probabilities 1/10, 2/10, 3/10, 4/10 respectively. Find E(X) and Var(X). Also calculate the mean and variance of the random variables (i) Y = 2X + 3 (ii) Z = sin(X).

Solution

E(X)

E(X) = (1)(1/10) + (2)(2/10) + (3)(3/10) + (4)(4/10) = 1/10 + 4/10 + 9/10 + 16/10 = 30/10 = 3

Var(X)

E(X^2) = (1^2)(1/10) + (2^2)(2/10) + (3^2)(3/10) + (4^2)(4/10) = 1/10 + 8/10 + 27/10 + 64/10 = 100/10 = 10

Var(X) = E(X^2) - (E(X))^2 = 10 - (3)^2 = 10 - 9 = 1

(i) Y = 2X + 3

E(Y) = E(2X + 3) = 2E(X) + 3 = 2(3) + 3 = 6 + 3 = 9

Var(Y) = Var(2X + 3) = 2^2 Var(X) = 4 * 1 = 4

(ii) Z = sin(X)

E(Z) = E(sin(X)) = (sin(1))(1/10) + (sin(2))(2/10) + (sin(3))(3/10) + (sin(4))(4/10)

E(Z) ≈ (0.8415)(1/10) + (0.9093)(2/10) + (0.1411)(3/10) + (-0.7568)(4/10)

E(Z) ≈ 0.08415 + 0.18186 + 0.04233 - 0.30272 = 0.00562

E(Z^2) = (sin^2(1))(1/10) + (sin^2(2))(2/10) + (sin^2(3))(3/10) + (sin^2(4))(4/10)

E(Z^2) ≈ (0.708)(1/10) + (0.8268)(2/10) + (0.0199)(3/10) + (0.5727)(4/10)

E(Z^2) ≈ 0.0708 + 0.16536 + 0.00597 + 0.22908 ≈ 0.47121

Var(Z) = E(Z^2) - (E(Z))^2 ≈ 0.47121 - (0.00562)^2 ≈ 0.47121 - 0.00003158 ≈ 0.471178

Question 4

Two unbiased dice are thrown simultaneously. Find the expectation and variance of the sum of the numbers appearing on the dice.

Solution

Let X be the sum of the numbers on the two dice. The possible values of X range from 2 to 12.

E(X) = E(Die 1) + E(Die 2) = (1+2+3+4+5+6)/6 + (1+2+3+4+5+6)/6 = 3.5 + 3.5 = 7

Var(Die 1) = Var(Die 2) = E(X^2) - E(X)^2

E(X^2) for a single die = (1^2 + 2^2 + 3^2 + 4^2 + 5^2 + 6^2) / 6 = (1 + 4 + 9 + 16 + 25 + 36) / 6 = 91 / 6

Var(Die 1) = Var(Die 2) = 91/6 - (3.5)^2 = 91/6 - 49/4 = (182 - 147) / 12 = 35/12

Var(X) = Var(Die 1) + Var(Die 2) = 35/12 + 35/12 = 70/12 = 35/6

Question 5

The cumulative distribution function of a random variable X is given by...

F(x) = { 0, x < 0; 0.05, 0 ≤ x < 1; 0.20, 1 ≤ x < 2; 0.45, 2 ≤ x < 3; 0.70, 3 ≤ x < 4; 1, x ≥ 4 }

(a) Find the mean and Variance of X.

First, find the probability mass function (PMF):

  • P(X = 0) = F(0) - F(0-) = 0.05 - 0 = 0.05
  • P(X = 1) = F(1) - F(1-) = 0.20 - 0.05 = 0.15
  • P(X = 2) = F(2) - F(2-) = 0.45 - 0.20 = 0.25
  • P(X = 3) = F(3) - F(3-) = 0.70 - 0.45 = 0.25
  • P(X = 4) = F(4) - F(4-) = 1 - 0.70 = 0.30

Mean (E[X]) = (0)(0.05) + (1)(0.15) + (2)(0.25) + (3)(0.25) + (4)(0.30) = 0 + 0.15 + 0.50 + 0.75 + 1.20 = 2.6

E[X^2] = (0^2)(0.05) + (1^2)(0.15) + (2^2)(0.25) + (3^2)(0.25) + (4^2)(0.30) = 0 + 0.15 + 1 + 2.25 + 4.8 = 8.2

Variance (Var[X]) = E[X^2] - (E[X])^2 = 8.2 - (2.6)^2 = 8.2 - 6.76 = 1.44

(b) If Y = (X – 2)(X – 3)(X – 4), find the probability distribution of Y. Hence or otherwise, find E[Y] and var[Y].

Calculate Y for each X:

  • If X = 0, Y = (0 - 2)(0 - 3)(0 - 4) = (-2)(-3)(-4) = -24
  • If X = 1, Y = (1 - 2)(1 - 3)(1 - 4) = (-1)(-2)(-3) = -6
  • If X = 2, Y = (2 - 2)(2 - 3)(2 - 4) = (0)(-1)(-2) = 0
  • If X = 3, Y = (3 - 2)(3 - 3)(3 - 4) = (1)(0)(-1) = 0
  • If X = 4, Y = (4 - 2)(4 - 3)(4 - 4) = (2)(1)(0) = 0

Probability Distribution of Y:

  • P(Y = -24) = P(X = 0) = 0.05
  • P(Y = -6) = P(X = 1) = 0.15
  • P(Y = 0) = P(X = 2) + P(X = 3) + P(X = 4) = 0.25 + 0.25 + 0.30 = 0.80

E[Y] = (-24)(0.05) + (-6)(0.15) + (0)(0.80) = -1.2 - 0.9 + 0 = -2.1

E[Y^2] = (-24)^2(0.05) + (-6)^2(0.15) + (0)^2(0.80) = 576(0.05) + 36(0.15) + 0 = 28.8 + 5.4 = 34.2

Var[Y] = E[Y^2] - (E[Y])^2 = 34.2 - (-2.1)^2 = 34.2 - 4.41 = 29.79

Question 6

Consider the experiment of tossing a pair of fair dice and let X denote the minimum of the numbers shown. Find the pmf of X and also of E(X) and Var(X).

Solution

PMF of X

Possible values for X (minimum) are 1, 2, 3, 4, 5, 6.

  • P(X = 1): Occurs when at least one die is 1. Favorable outcomes are (1,1), (1,2), (1,3), (1,4), (1,5), (1,6), (2,1), (3,1), (4,1), (5,1), (6,1). Total 11 outcomes. P(X=1) = 11/36
  • P(X = 2): Occurs when the minimum is 2, i.e., both dice are at least 2 and at least one is 2. Outcomes are (2,2), (2,3), (2,4), (2,5), (2,6), (3,2), (4,2), (5,2), (6,2). Total 9 outcomes. P(X = 2) = 9/36.
  • P(X = 3): Outcomes are (3,3), (3,4), (3,5), (3,6), (4,3), (5,3), (6,3). Total 7 outcomes. P(X = 3) = 7/36.
  • P(X = 4): Outcomes are (4,4), (4,5), (4,6), (5,4), (6,4). Total 5 outcomes. P(X=4) = 5/36.
  • P(X = 5): Outcomes are (5,5), (5,6), (6,5). Total 3 outcomes. P(X=5) = 3/36.
  • P(X = 6): Outcome is (6,6). Total 1 outcome. P(X = 6) = 1/36.

E(X)

E(X) = (1)(11/36) + (2)(9/36) + (3)(7/36) + (4)(5/36) + (5)(3/36) + (6)(1/36)

E(X) = (11 + 18 + 21 + 20 + 15 + 6) / 36 = 91/36 ≈ 2.5278

Var(X)

E(X^2) = (1^2)(11/36) + (2^2)(9/36) + (3^2)(7/36) + (4^2)(5/36) + (5^2)(3/36) + (6^2)(1/36)

E(X^2) = (11 + 36 + 63 + 80 + 75 + 36) / 36 = 301/36 ≈ 8.3611

Var(X) = E(X^2) - (E(X))^2 = 301/36 - (91/36)^2 = 301/36 - 8281/1296 = (10836 - 8281) / 1296 = 2555/1296 ≈ 1.9715

Question 7

A box contains 10 items, of which 4 are defective. Suppose that 3 items are drawn from the box without replacement, and let X denote the number of defective items drawn. (i) Find the probability distribution of X (ii) What is the probability that at most one defective item would be found among the drawn items? (iii) Find the mean and variance of X.

Solution

(i) Probability Distribution of X

X can take values 0, 1, 2, 3.

Total number of ways to choose 3 items from 10: C(10, 3) = 120

  • P(X = 0): C(6, 3) / C(10, 3) = 20/120 = 1/6
  • P(X = 1): (C(4, 1) * C(6, 2)) / C(10, 3) = 60/120 = 1/2
  • P(X = 2): (C(4, 2) * C(6, 1)) / C(10, 3) = 36/120 = 3/10
  • P(X = 3): (C(4, 3) * C(6, 0)) / C(10, 3) = 4/120 = 1/30

(ii) Probability of at most one defective item

P(X ≤ 1) = P(X = 0) + P(X = 1) = 1/6 + 1/2 = 4/6 = 2/3

(iii) Mean and Variance of X

E(X) = (0)(1/6) + (1)(1/2) + (2)(3/10) + (3)(1/30) = 1.2

Var(X) = E(X^2) - (E(X))^2 = 2 - (1.2)^2 = 2 - 1.44 = 0.56

Question 8

A fair coin is tossed until a head appears. Let X denote the number of tosses.

Solution

(a) Probability Mass Function (PMF) and Distribution Function

This is a geometric distribution.

PMF: P(X = k) = (1/2)^k, for k = 1, 2, 3, ...

CDF: F(x) = P(X ≤ x) = 1 - (1/2)^x, for x = 1, 2, 3, ...

(b) Probability that the number of tosses is even/odd

P(X is even) = P(X = 2) + P(X = 4) + ... = (1/2)^2 + (1/2)^4 + ... = 1/3

P(X is odd) = 1 - P(X is even) = 1 - 1/3 = 2/3

(c) Average Number of Tosses Until a Head Appears

The average number of tosses is the expected value of the geometric distribution, E[X] = 1/p, where p = 1/2.
